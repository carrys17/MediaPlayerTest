# MediaPlayerTest

该项目主要是设置关于音乐特效的属性，可以显示音乐示波器、设置均衡控制器、设置重低音、设置预设音场。

主要是MediaPlayer与 Visualizer(示波器)、Equalizer(均衡控制器)、BassBoost(重低音控制器)、PresetReverb(预设音场控制器)结合使用。

其中示波器有三种效果，块状波形图、柱状波形图和曲线波形图
